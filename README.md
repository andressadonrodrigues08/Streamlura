# 🎬 Streamlura

Projeto desenvolvido durante uma Imersão.

## 📚 Sobre o projeto

O Streamlura é uma aplicação web inspirada em plataformas de streaming, onde é possível visualizar conteúdos organizados de forma moderna e interativa.

Este projeto foi criado com foco em praticar conceitos de front-end, aplicando na prática o desenvolvimento de interfaces web.

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

## ✨ Funcionalidades

- Layout inspirado em plataformas de streaming  
- Organização de conteúdos em seções  
- Interações básicas com JavaScript  
- Interface visual moderna
